Keys stay on this device in chrome.storage.local. Do not publish a build that embeds your key; use a backend proxy if you ship to others.

OpenAI

Stored only in this browser profile.

If set, the extension POSTs { word, context, targetLang, includeDefinition } to your server instead of calling OpenAI directly. Your server should add the secret and call OpenAI.

Translation

On by default. The model always returns a short gloss for word lookups (including phrasal verbs when relevant). Turn this off to hide that line and save a little vertical space.
